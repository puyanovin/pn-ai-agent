Open-source AI Assistant for WordPress.

## Features

- Multiple AI Providers
- OpenAI
- Gemini
- Claude
- DeepSeek
- Qwen
- OpenAI Compatible APIs
- Chat Widget
- Chat Shortcode
- Gutenberg Block
- Chat History
- Logs
- Provider Test
- Multilingual
- RTL Support

## Installation

```bash
composer install
```

Activate the plugin inside WordPress.

## Requirements

- WordPress 6.5+
- PHP 8.1+
