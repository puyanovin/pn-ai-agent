# Contributing

Thank you for your interest in PN AI Agent.

We welcome contributions from the community.

## Development

Clone the repository

```bash
git clone https://github.com/puyanovin/pn-ai-agent.git
```

Install dependencies

```bash
composer install
```

Run WordPress locally and activate the plugin.

## Coding Standards

- Follow WordPress Coding Standards
- Use PSR-12 for PHP
- Escape all output
- Sanitize all input
- Keep code modular

## Pull Requests

Please:

- Create a feature branch
- Write clear commit messages
- Test before submitting
- Update CHANGELOG when needed
