== Description ==

PN AI Agent is an open-source AI Assistant plugin for WordPress.

It allows you to connect different AI providers and place an intelligent chatbot anywhere on your website.

Features:

* Multiple AI Providers
* OpenAI
* Google Gemini
* Anthropic Claude
* DeepSeek
* Qwen
* OpenAI Compatible APIs
* Chat Widget
* Chat Shortcode
* Gutenberg Block
* Provider Connection Test
* Chat History
* Fully Translatable
* Lightweight
* Fast

== Installation ==

1. Upload the plugin to /wp-content/plugins/
2. Activate the plugin.
3. Go to PN AI Agent.
4. Configure your AI Provider.
5. Add the chatbot using the shortcode or widget.
